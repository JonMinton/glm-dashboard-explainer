Choose the Link Function

Blood pressure is strictly positive (min = 94 mm Hg in our data). Which link function ensures our predictions will always be positive?

Log Link

$g(\mu) = \log(\mu)$

Predictions via $\mu = e^{\eta}$ are always positive. Coefficients give multiplicative (percentage) effects.

Inverse Link

$g(\mu) = \frac{1}{\mu}$

The canonical link for Gamma. Predictions positive if $\eta > 0$, but harder to interpret.

Identity Link

$g(\mu) = \mu$

Predictions equal $\eta$ directly. Simple additive interpretation, but no constraint on sign.

Logit Link

$g(\mu) = \log\left(\frac{\mu}{1-\mu}\right)$

Maps $\mu$ to $(-\infty, \infty)$, constrains $\mu$ to (0,1). For probabilities only.
