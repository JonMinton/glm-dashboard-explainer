Choose the Link Function

Blood pressure is strictly positive (min = 94 mm Hg in our data). Which link function ensures our predictions will always be positive?

Log Link

$g(\mu) = \log(\mu)$

Predictions via $\mu = e^{\eta}$ are always positive. Coefficients give multiplicative (percentage) effects.

Inverse Link

$g(\mu) = \frac{1}{\mu}$

The canonical link for Gamma. Predictions positive if $\eta > 0$, but harder to interpret.

Identity Link

$g(\mu) = \mu$

Predictions equal $\eta$ directly. Simple additive interpretation, but no constraint on sign.

Logit Link

$g(\mu) = \log\left(\frac{\mu}{1-\mu}\right)$

Maps $\mu$ to $(-\infty, \infty)$, constrains $\mu$ to (0,1). For probabilities only.

✓ Log Link Selected

The log link ensures positive predictions and gives interpretable multiplicative effects.

✔ Correct! Log link ensures positive predictions.

The log link is the best choice for positive continuous data because:

Inverse link function: $\mu = e^{\eta} = e^{\beta_0 + \beta_1 X_1 + ...}$

Since $e^x > 0$ for all $x$, predictions are always positive!

Bonus: Coefficients have a multiplicative interpretation:

  • $e^{\beta}$ gives the multiplicative effect on the mean
  • $e^{0.004} \approx 1.004$ means each year of age multiplies BP by 1.004 (â‰ˆ +0.4%)
  • Much more intuitive than additive effects for many positive outcomes

Note: Log is not the canonical link for Gamma (inverse is), but log is more commonly used because of interpretability.

⚠ Valid but rarely used in practice.

The inverse link is actually the canonical link for the Gamma distribution. This means it has nice mathematical properties for estimation.

Inverse link: $\mu = \frac{1}{\eta}$

Only positive if $\eta > 0$, which isn't guaranteed.

Problems with inverse link:

  • Predictions can be negative if $\eta < 0$
  • Coefficients are hard to interpret (reciprocal relationships)
  • Model can be unstable near $\eta = 0$

In practice, the log link is almost always preferred for Gamma regression because it guarantees positive predictions and gives interpretable multiplicative effects.

❌ Wrong! Identity link allows negative predictions.

With the identity link, predictions equal the linear predictor directly:

$\mu = \eta = \beta_0 + \beta_1 X_1 + ...$

This can easily be negative!

Example: If $\beta_0 = 100$ and $\beta_1 = -5$, then for someone with $X_1 = 30$:

$\mu = 100 + (-5)(30) = 100 - 150 = -50$ mm Hg

Negative blood pressure is impossible! For positive continuous outcomes, we need a link function that constrains predictions to be positive.

❌ Wrong! Logit is for probabilities, not continuous data.

The logit link constrains $\mu$ to the interval (0, 1):

$\mu = \frac{e^{\eta}}{1 + e^{\eta}}$ is always between 0 and 1

Problem: Blood pressure ranges from 94 to 200 mm Hg in our data, not 0 to 1!

The logit link is designed for binary outcomes (Bernoulli/Binomial) where we're modelling a probability. For positive continuous outcomes like blood pressure, we need a link that allows any positive value.
