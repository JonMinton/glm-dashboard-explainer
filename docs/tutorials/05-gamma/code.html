Gamma GLM Implementation

Let's fit our Gamma model to predict resting blood pressure using the UCI Heart Disease dataset. 

# Load UCI Heart Disease data
heart <- read.csv(
  "https://archive.ics.uci.edu/ml/machine-learning-databases/heart-disease/processed.cleveland.data",
  header = FALSE, na.strings = "?"
)
names(heart) <- c("age", "sex", "cp", "trestbps", "chol", "fbs",
                  "restecg", "thalach", "exang", "oldpeak",
                  "slope", "ca", "thal", "target")
heart <- heart[complete.cases(heart), ]

# Fit Gamma GLM with log link
fit_gamma <- glm(trestbps ~ age + exang + oldpeak,
                 data = heart,
                 family = Gamma(link = "log"))

# View results
summary(fit_gamma)

# Shape parameter (1/dispersion)
shape <- 1 / summary(fit_gamma)$dispersion
cat("Shape parameter:", shape, "\n")

# Compare with Gaussian
fit_gauss <- glm(trestbps ~ age + exang + oldpeak,
                 data = heart,
                 family = gaussian())
cat("Gamma AIC:", AIC(fit_gamma), "\n")
cat("Gaussian AIC:", AIC(fit_gauss), "\n")

import pandas as pd
import numpy as np
import statsmodels.api as sm
import statsmodels.formula.api as smf

# Load UCI Heart Disease data
url = "https://archive.ics.uci.edu/ml/machine-learning-databases/heart-disease/processed.cleveland.data"
cols = ["age", "sex", "cp", "trestbps", "chol", "fbs",
        "restecg", "thalach", "exang", "oldpeak",
        "slope", "ca", "thal", "target"]
heart = pd.read_csv(url, header=None, names=cols, na_values="?")
heart = heart.dropna()

# Fit Gamma GLM with log link
fit_gamma = smf.glm(
    'trestbps ~ age + exang + oldpeak',
    data=heart,
    family=sm.families.Gamma(link=sm.families.links.Log())
).fit()

# View results
print(fit_gamma.summary())

# Shape parameter
shape = 1 / fit_gamma.scale
print(f"\nShape parameter: {shape:.2f}")

# Compare with Gaussian
fit_gauss = smf.glm(
    'trestbps ~ age + exang + oldpeak',
    data=heart,
    family=sm.families.Gaussian()
).fit()
print(f"Gamma AIC: {fit_gamma.aic:.1f}")
print(f"Gaussian AIC: {fit_gauss.aic:.1f}")

Multiplicative Interpretation (Log Link)

With the log link, $e^{\beta}$ gives the multiplicative effect on blood pressure:

  • Age: $e^{0.004} = 1.004$ → Each additional year multiplies BP by 1.004
  • A 10-year increase: $1.004^{10} = 1.041$ → BP increases by ~4.1%
  • Oldpeak: $e^{0.0155} = 1.016$ → Each unit of ST depression multiplies BP by 1.016

Key Takeaway

The Gamma model wins on AIC by ~16 points and guarantees positive predictions. For strictly positive outcomes like blood pressure, Gamma with log link is the natural choice.
