A New Type of Outcome: Counts

Unlike Tutorial 1 (continuous heart rate) or Tutorial 2 (binary disease status), here we're predicting a count: how many bikes are rented each day (0, 1, 2, ..., thousands). Counts are always non-negative integers—this requires a different type of GLM.

Step 1: Select the Response Variable

Drag Rental Count to the Response zone on the right.

The response variable (y) is what we're trying to predict.

Why this variable? We want to predict how many bikes will be rented based on weather conditions.
