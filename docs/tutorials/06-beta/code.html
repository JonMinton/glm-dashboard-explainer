Beta Regression Implementation

Let's fit our Beta regression model to predict exam proportions. Note that Beta regression requires specialised packages — it is not available in base R's glm() or Python's default statsmodels.glm. 

# Install if needed: install.packages("betareg")
library(betareg)

# Load data
exam <- read.csv("exam-proportions.csv")

# Fit Beta regression
fit_beta <- betareg(proportion_correct ~ study_hours + prior_gpa + has_tutor,
                    data = exam)
summary(fit_beta)

import pandas as pd
import statsmodels.api as sm
from statsmodels.othermod.betareg import BetaModel

# Load data
exam = pd.read_csv("exam-proportions.csv")

# Fit Beta regression
X = sm.add_constant(exam[['study_hours', 'prior_gpa', 'has_tutor']])
model = BetaModel(exam['proportion_correct'], X)
fit = model.fit()
print(fit.summary())

A note on the precision row:

By default, statsmodels models the precision on the log scale, so the precision coefficient is $\ln\hat\phi = 3.418$, giving $\hat\phi = e^{3.418} \approx 30.5$ — the same value R's betareg reports directly (it uses an identity link for $\phi$ by default).

Interpreting Logit-Scale Coefficients

With the logit link, coefficients are on the log-odds scale, just like logistic regression. To get the effect on the proportion scale, you need the inverse logit (logistic function):

  • Example: For a student with 10 study hours, GPA 3.0, no tutor:
  • $\eta = -3.656 + 0.121 \times 10 + 1.043 \times 3.0 + 0.384 \times 0 = 0.68$
  • $\mu = \frac{1}{1 + e^{-0.68}} = 0.66$ (predicted 66% correct)

Effect of Tutoring

Practical Example

For two students with identical study hours (10) and GPA (3.0):

  • Without tutor: $\eta = 0.68$, $\mu = 0.66$ (66% correct)
  • With tutor: $\eta = 0.68 + 0.384 = 1.07$, $\mu = 0.74$ (74% correct)
  • Tutoring effect: +8 percentage points for this student profile

Note: The effect on the proportion scale varies depending on where you are on the curve. Near 0.5, the effect is largest; near 0 or 1, it's compressed.
