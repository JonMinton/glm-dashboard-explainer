How Beta Regression Fitting Works

Beta regression uses maximum likelihood estimation (MLE) to find the best-fitting parameters. Unlike the standard GLM families (Gaussian, Binomial, Poisson, Gamma), the Beta distribution is not part of the classical exponential family used by R's glm() function — so it requires a separate package.

The Precision Parameter

The Beta distribution has two shape parameters ($a$ and $b$), but for regression we reparameterise in terms of the mean $\mu$ and precision $\phi$:

$a = \mu \phi, \quad b = (1-\mu) \phi$

Variance formula: $\text{Var}(Y) = \frac{\mu(1-\mu)}{1 + \phi}$

Mean-Precision Parameterisation

The standard Beta distribution uses shape parameters $a$ and $b$. For regression, we reparameterise:

$\mu = \frac{a}{a+b}, \quad \phi = a + b$

This is analogous to how we handle other GLM families:

  • $\mu$ is modelled via the systematic component: $g(\mu) = X\beta$
  • $\phi$ is the precision (inverse of dispersion), estimated separately

Key Insight: Precision $\phi$ Plays the Same Role as Other Dispersion Parameters

Just as $\sigma^2$ controls dispersion in Gaussian, and shape controls dispersion in Gamma, the precision parameter $\phi$ controls how tightly data clusters around the mean in Beta regression.

Higher $\phi$ = less variability (more precise predictions). Lower $\phi$ = more spread around the mean.

MLE for Beta Regression

Since Beta regression is not in base R's glm(), it uses direct maximum likelihood optimisation rather than IRLS. The betareg package in R (and statsmodels.othermod.betareg in Python) handles this automatically.

The optimisation simultaneously estimates:

  1. Mean model coefficients $\beta$: How predictors affect the expected proportion
  2. Precision $\phi$: How tightly observations cluster around the predicted mean

Comparison: All GLM Families

Family Variance Dispersion Common link
Gaussian $\sigma^2$ $\sigma^2$ (estimated) Identity
Binomial $\mu(1-\mu)$ 1 (fixed) Logit
Poisson $\mu$ 1 (fixed) Log
Neg. Binomial $\mu + \mu^2/\theta$ $\theta$ (estimated) Log
Gamma $\mu^2/\text{shape}$ $1/\text{shape}$ (estimated) Log
Beta $\mu(1-\mu)/(1+\phi)$ $\phi$ (estimated) Logit

For Our Exam Proportions Model

After fitting, we get a precision of approximately:

$\hat{\phi} \approx 30.5$

This means for a predicted proportion of $\mu = 0.5$:

$\text{SD} = \sqrt{\frac{0.5 \times 0.5}{1 + 30.5}} \approx 0.089$

Interpretation

A precision of 30.5 means that for a student predicted to score 50% correct, the standard deviation is about 8.9 percentage points. For a student predicted to score 80%, the SD is smaller: $\sqrt{0.8 \times 0.2 / 31.5} \approx 0.071$ (7.1 pp).

Notice: Unlike Gaussian, the variance automatically decreases near the boundaries (0 and 1) — a natural property of the Beta distribution.
