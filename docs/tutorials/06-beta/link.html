Choose the Link Function

Exam proportions are bounded between 0 and 1. Which link function maps this bounded interval to the entire real line?

Logit Link

$g(\mu) = \log\left(\frac{\mu}{1-\mu}\right)$

Maps $\mu \in (0,1)$ to $\eta \in (-\infty, +\infty)$. Inverse: $\mu = \frac{1}{1+e^{-\eta}}$.

Identity Link

$g(\mu) = \mu$

No transformation. Predictions equal the linear predictor directly.

Log Link

$g(\mu) = \log(\mu)$

Maps $(0, \infty)$ to $(-\infty, +\infty)$. Ensures positive predictions.

Probit Link

$g(\mu) = \Phi^{-1}(\mu)$

Inverse normal CDF. Maps $(0,1)$ to $(-\infty, +\infty)$.

✓ Logit Link Selected

The logit link maps (0,1) to the real line, keeping predictions bounded as proportions.

✔ Correct! Logit maps (0,1) to the real line.

The logit link is the same link function you saw in logistic regression (Tutorial 2). The key difference is what happens after the link:

Inverse link function: $\mu = \frac{1}{1 + e^{-\eta}}$

Since the logistic function always outputs values in $(0,1)$, predictions are always valid proportions!

Key insight: The link function handles the constraint — the distribution handles the shape.

  • Logistic regression: Logit link + Binomial distribution (for binary/count data)
  • Beta regression: Logit link + Beta distribution (for continuous proportions)

Same link, different distributions — because the data-generating processes are fundamentally different.

❌ Wrong! Identity link allows predictions outside (0,1).

With the identity link, predictions equal the linear predictor directly:

$\mu = \eta = \beta_0 + \beta_1 X_1 + ...$

This can easily exceed the (0,1) boundaries!

Example: If $\beta_0 = 0.5$ and $\beta_1 = 0.1$, then for a student with 20 study hours:

$\mu = 0.5 + 0.1 \times 20 = 2.5$

A proportion of 2.5 is nonsensical! Proportions must lie between 0 and 1. We need a link function that respects these boundaries.

❌ Wrong! Log link allows values greater than 1.

The log link maps $(0, \infty)$ to $(-\infty, +\infty)$:

$\mu = e^{\eta}$ is always positive, but can exceed 1!

Problem: The log link prevents negative predictions but allows values greater than 1. A predicted proportion of 2.5 makes no sense for exam performance.

The log link is great for positive unbounded data (Poisson, Gamma), but for data bounded between 0 and 1, we need a link that constrains to the unit interval.

⚠ Valid alternative, but logit is more common.

The probit link is a valid alternative that also maps $(0,1)$ to $(-\infty, +\infty)$. It uses the inverse normal CDF:

$g(\mu) = \Phi^{-1}(\mu)$, so $\mu = \Phi(\eta)$

Like logit, this constrains predictions to $(0,1)$.

In practice: Results are nearly identical to logit. The logit link is preferred because:

  • Coefficients have an odds-ratio interpretation
  • More widely used and understood
  • Computationally slightly simpler

Both are valid — but for this tutorial we'll use the logit link (the standard choice for Beta regression).
